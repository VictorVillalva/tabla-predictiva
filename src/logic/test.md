# Tabla Predictiva

Este README cuenta con cadenas de ejemplo las cuales son validadas por este programa

 * Example
| automata alfabeto : (a...z/0...9) , (a...z/0...9) ; aceptacion : (0...9) ; fin
| automata alfabeto : (a...z/0...9) , (a...z/0...9) , n(a...z/0...9) ; aceptacion : (0...9) ; fin
